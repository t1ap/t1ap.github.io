TDPE Shaders

Latest version: 1.5



TDPE Shaders is a Shader pack for Minecraft PE that aims at making the graphics of the game more realistic and beautiful. This shader improves numerous graphical aspects of the game, but if you don't like any of the features, you can turn them off using shader settings.

  This shader may not work on Windows 10 edition

Features:

  • Water Waves
  • Underwater distortion
  • Underwater Effect
  • Underwater Sun beams
  • Sun beams
  • Sun Reflection
  • Player Shadow
  • Rain Effect
  • Bump Effect (Normal Mapping)
  • End Galaxy effect
  • End streaks
  • Sun rays
  • POM
  • Glow
  • Shader Settings (which can be found in TDPE Shaders/shaders/glsl/Shader_Settings/shadersettings.h)

Changelog
  • Added POM (Turned off by default. To turn it on, go to shader settings and set PARALLAX_MAPPING and BUMP_EFFECT to 1)
    • (If the POM looks glitchy on your device, try setting POM_TBN_SIMPLE to 1)
  • Added a glow? effect to light sources
  • Optimizations to Player Shadow
This shader currently doesn't work on Render Dragon.
